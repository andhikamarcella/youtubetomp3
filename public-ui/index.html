Lisensi & Kepatuhan

Alat ini dimaksudkan untuk mengonversi konten yang Anda miliki atau yang secara tegas diizinkan oleh pemilik hak cipta. Mengunduh atau mendistribusikan konten tanpa izin dapat melanggar hukum dan ketentuan layanan platform. Anda bertanggung jawab penuh atas penggunaan Anda.

Fix "Failed to fetch" — Siapkan Backend

Error Failed to fetch muncul karena endpoint /api/convert belum tersedia atau terblokir CORS/HTTPS. Ikuti panduan singkat ini.

  1. Jalankan server backend di domain/port yang sama dengan frontend, atau aktifkan CORS.
  2. Pastikan HTTPS cocok: jika halaman dibuka lewat https://, backend juga harus https:// (hindari mixed content).
  3. Periksa path: UI memanggil POST /api/convert. Jika backend di subpath lain, ubah URL di JS.
  4. Periksa log di DevTools (F12 → Network) untuk status, CORS header, dan pesan error.

Contoh Backend: Node.js + Express (+ yt-dlp + FFmpeg)

Contoh ini mengonversi audio YouTube ke MP3 (untuk konten yang Anda miliki/diizinkan). Simpan file di folder server/.

// server/package.json
{
  
  "name": "ytmp3-backend",
  "type": "module",
  "scripts": {
    "start": "node index.js"
  },
  "dependencies": {
    "cors": "^2.8.5",
    "express": "^4.19.2",
    "nanoid": "^5.0.7"
  }
}

// server/.env (opsional)
PORT=3000

// server/index.js
import express from 'express';
import cors from 'cors';
import { spawn } from 'child_process';
import { fileURLToPath } from 'url';
import { dirname, join } from 'path';
import { existsSync, mkdirSync } from 'fs';
import { nanoid } from 'nanoid';

const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);

const app = express();
app.use(express.json({ limit: '1mb' }));
app.use(cors()); // jika frontend beda origin. Jika satu origin, Anda boleh hapus ini.

// Sediakan folder publik untuk hasil unduhan
const PUBLIC_DIR = join(__dirname, 'public');
const JOBS_DIR = join(PUBLIC_DIR, 'jobs');
if (!existsSync(PUBLIC_DIR)) mkdirSync(PUBLIC_DIR);
if (!existsSync(JOBS_DIR)) mkdirSync(JOBS_DIR);
app.use('/jobs', express.static(JOBS_DIR, { fallthrough: false }));

// Cek ketersediaan yt-dlp & ffmpeg di PATH
function checkTool(cmd) {
  return new Promise((resolve) => {
    const p = spawn(cmd, ['-version']);
    p.on('error', () => resolve(false));
    p.on('close', (code) => resolve(code === 0));
  });
}

app.post('/api/convert', async (req, res) => {
  try {
    const { url, quality = 128, id3, trim, normalize } = req.body || {};
    if (!url) return res.status(400).json({ error: 'Missing url' });

    const hasYtDlp = await checkTool('yt-dlp');
    const hasFfmpeg = await checkTool('ffmpeg');
    if (!hasYtDlp || !hasFfmpeg) {
      return res.status(500).json({ error: 'yt-dlp/ffmpeg tidak ditemukan di server' });
    }

    const id = nanoid(10);
    const jobDir = join(JOBS_DIR, id);
    mkdirSync(jobDir);

    // 1) Ambil audio terbaik (m4a/webm) dari YouTube
    const bestAudio = join(jobDir, 'audio.m4a');
    const ytdlpArgs = [
      '-f', 'bestaudio/best',
      '-o', bestAudio,
      url
    ];

    await new Promise((resolve, reject) => {
      const y = spawn('yt-dlp', ytdlpArgs);
      y.on('error', reject);
      y.stderr.on('data', d => process.stderr.write(d));
      y.on('close', code => code === 0 ? resolve() : reject(new Error('yt-dlp gagal')));
    });

    // 2) Konversi ke MP3 sesuai kualitas + opsi trim/normalize
    const outFile = join(jobDir, 'output.mp3');
    const ffArgs = ['-y', '-i', bestAudio];

    if (trim?.start) ffArgs.unshift('-ss', trim.start);
    if (trim?.end) ffArgs.push('-to', trim.end);

    if (normalize) {
      // filter loudness dasar
      ffArgs.push('-af', 'dynaudnorm');
    }

    ffArgs.push('-vn', '-codec:a', 'libmp3lame', '-b:a', `${quality}k`, outFile);

    await new Promise((resolve, reject) => {
      const f = spawn('ffmpeg', ffArgs);
      f.on('error', reject);
      f.stderr.on('data', d => process.stderr.write(d));
      f.on('close', code => code === 0 ? resolve() : reject(new Error('ffmpeg gagal')));
    });

    // 3) (Opsional) tulis ID3 dengan ffmpeg -metadata
    if (id3?.title || id3?.artist) {
      const tagged = join(jobDir, 'output.tagged.mp3');
      const meta = [];
      if (id3.title) meta.push('-metadata', `title=${id3.title}`);
      if (id3.artist) meta.push('-metadata', `artist=${id3.artist}`);
      await new Promise((resolve, reject) => {
        const f2 = spawn('ffmpeg', ['-y', '-i', outFile, ...meta, '-codec:a', 'copy', tagged]);
        f2.on('error', reject);
        f2.on('close', code => code === 0 ? resolve() : reject(new Error('penulisan ID3 gagal')));
      });
      // ganti file
      await new Promise((resolve, reject) => {
        const f3 = spawn(process.platform === 'win32' ? 'cmd' : 'bash', process.platform === 'win32' ? ['/c', 'move', '/y', tagged, outFile] : ['-lc', `mv -f "${tagged}" "${outFile}"`]);
        f3.on('error', reject);
        f3.on('close', () => resolve());
      });
    }

    const filename = `${(id3?.title || 'audio')}-${quality}kbps.mp3`;
    const downloadUrl = `/jobs/${id}/output.mp3`;
    res.json({ filename, downloadUrl });
  } catch (e) {
    console.error(e);
    res.status(500).json({ error: e.message || 'Server error' });
  }
});

// (Opsional) layani file frontend statis jika ditempatkan di ../public-ui
// app.use(express.static(join(__dirname, '../public-ui')))

const port = process.env.PORT || 3000;
app.listen(port, () => console.log('Backend listening on :' + port));

Instalasi Cepat

# 1) Siapkan tool sistem
# Windows: winget install Gyan.FFmpeg; choco install yt-dlp
# macOS: brew install ffmpeg yt-dlp
# Linux: sudo apt-get install -y ffmpeg && pipx install yt-dlp

# 2) Jalankan backend
cd server
npm i
npm start

# 3) Pastikan frontend memanggil origin yang sama
# Jika UI Anda di http://localhost:3000, taruh file HTML ini lewat Express static
# atau ubah fetch('/api/convert') menjadi fetch('http://localhost:3000/api/convert').

Checklist CORS/HTTPS

  • Jika UI di file:// atau domain berbeda, aktifkan cors() di Express.
  • Jangan campur https (UI) dengan http (backend). Gunakan reverse proxy (Nginx) atau deploy di satu domain.
  • Uji dengan Postman/curl ke POST /api/convert untuk memastikan server OK.
Catatan legal: Gunakan hanya untuk materi yang Anda miliki haknya atau telah diizinkan oleh pemilik. Hormati ToS platform.

Deploy Backend di Railway (dengan FFmpeg & yt‑dlp)

Railway cocok buat backend Node/Express kita. Kita pakai Dockerfile biar FFmpeg & yt-dlp tersedia.

1) Struktur repo

your-repo/
├─ server/
│  ├─ index.js        # backend (sudah ada di canvas)
│  ├─ package.json    # sudah ada di canvas
├─ Dockerfile
├─ railway.toml

2) Dockerfile

# Dockerfile
FROM node:20-slim

# Install ffmpeg & python3-pip untuk yt-dlp
RUN apt-get update \
    && apt-get install -y --no-install-recommends ffmpeg python3-pip \
    && pip3 install --no-cache-dir yt-dlp \
    && rm -rf /var/lib/apt/lists/*

# App files
WORKDIR /app
COPY server ./server
WORKDIR /app/server
RUN npm ci || npm i

# Expose port dari Railway
ENV PORT=8080

# Buat folder untuk file hasil
RUN mkdir -p public/jobs

# Start
CMD ["node", "index.js"]

3) railway.toml (Volume untuk hasil unduhan)

Kita pakai volume agar folder public/jobs persisten.

# railway.toml
[build]
builder = "DOCKERFILE"

[[volumes]]
name = "jobs"
mountPath = "/app/server/public/jobs"

4) Deploy

  1. Push repo ke GitHub (isi: server/index.js, server/package.json, Dockerfile, railway.toml).
  2. Buka Railway → New Project → Deploy from GitHub → pilih repo kamu.
  3. Tunggu build selesai. Service akan jalan di port $PORT (Railway berikan otomatis).
  4. Di tab Settings → Domains, aktifkan domain public (mis. https://ytmp3-yourapp.up.railway.app).

5) CORS & HTTPS

  • Backend contoh sudah pakai cors(). Jika perlu batasi, isi origin dengan domain InfinityFree kamu.
  • Railway domain sudah HTTPS → aman untuk dipanggil dari situs InfinityFree (yang juga HTTPS).

6) Hubungkan Frontend (InfinityFree atau file://)

// Di HTML (sudah ada variabel ini):
const API_BASE = 'https://ytmp3-yourapp.up.railway.app';

7) Debug

  • Logs: Railway → Service → Logs (cek error yt-dlp/ffmpeg).
  • Failed to fetch: cek mixed content (frontend https, backend https), dan CORS.
  • File tidak terunduh: pastikan link /jobs/<id>/output.mp3 diakses dan volume ter-mount.
Tip: Jika mau scale to zero hemat biaya, gunakan plan gratis dan aktifkan sleep. Pertanyaan? Kasih domain Railway kamu, nanti aku set API_BASE di file HTML kamu 😉
